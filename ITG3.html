Installation Guide

Follow these steps to set up In The Groove 3 - HD Version:

  1. Extract the ZIP File: After downloading the ZIP file, extract its contents into a new folder named In The Groove 3.
  2. Locate the Configuration File: Open the StepMania.ini file located in the following directory: In The Groove 3\Data.
  3. Configure the Settings: Adjust the following variables in the StepMania.ini file to match your system preferences: 
    DisplayHeight=1080                              // Set to your desired vertical resolution (e.g., 1080 for Full HD).
DisplayWidth=1920                              // Set to your desired horizontal resolution (e.g., 1920 for Full HD).
MemoryCardProfileImportSubdirs=In The Groove  // If this is blank, replace it with this
MemoryCardProfileSubdir=In The Groove 3      // Set this to "In The Groove 3" as shown
MemoryCardOsMountPointP1=D:\                // Replace "D:\" with the drive letter of Player 1's memory card.
MemoryCardOsMountPointP2=X:\               // Replace "X:\" with the drive letter of Player 2's memory card.
Windowed=0                                // Set to 1 if you prefer to run the game in a window instead of fullscreen.
  4. Notes: The Beta version may encounter crashes after the "Arcade Initialization" prompt. Please use at your own risk and report any issues.
