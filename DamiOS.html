DamiOS is a CMD-based operating system simulation under development. Explore this exciting project and its features!

Installation Guide

To install DamiOS manually, follow these steps:

# Open PowerShell and run the following commands:
irm https://bit.ly/Install-DamiOS

This may take several minutes.

When this is done, there should be a DamiOS icon on the Desktop. DamiOS is now installed

Ensure PowerShell script execution is enabled by running Set-ExecutionPolicy RemoteSigned as Administrator if needed.
