DamiOS - Installation

About DamiOS

DamiOS is a CMD-based operating system simulation under development, offering a fun and educational experience. It is designed for tech enthusiasts who enjoy experimenting with command-line tools and learning new technologies.

Step-by-Step Installation

  1. Open PowerShell as Administrator.
  2. Run the following command: 
    irm https://bit.ly/Install-DamiOS | iex
  3. Wait for the installation to complete.
  4. Ensure PowerShell script execution is enabled by running: 
    Set-ExecutionPolicy RemoteSigned
  5. Once complete, double-click the DamiOS icon on your desktop to start exploring!
