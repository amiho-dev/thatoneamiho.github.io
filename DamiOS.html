DamiOS Installation

Step 1: Open PowerShell

Open PowerShell as Administrator and execute the following command:

irm https://bit.ly/Install-DamiOS | iex

Step 2: Enable Execution

Ensure PowerShell script execution is enabled by running:

Set-ExecutionPolicy RemoteSigned

Step 3: Launch DamiOS

Once installation is complete, double-click the DamiOS icon on your desktop to start exploring!
